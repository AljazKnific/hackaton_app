## The Pipeline at a Glance

```
INTAKE → BUILD BRAIN (ICPOffer) → GENERATE (Ideas · Emotions · ViralHooks
       · WritingSystem · CTA · Voice · Structure-Formatting) → PICK → VOICE (ElevenLabs) → HANDOFF
```

## The 8 Folders
ICPOffer (brain) · Ideas (angles) · Emotions (triggers) · ViralHooks (hooks) · WritingSystem (copy craft) · CTA (the ask) · Voice (tone/VO) · Structure-Formatting (script layout)

## The Standard
Every promo: one ICP, one idea, outcome-led, honest, one CTA, built to be heard. Make the viewer feel *"that's my problem — and that's the fix."*
