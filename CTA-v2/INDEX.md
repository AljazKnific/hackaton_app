# CTA-v2 — Index (App CTA Engine)

How the app writes the closing ask of every promo. Feeds the **CTA beat (22–30s)** of Stage 3 scripts.

**Version:** 2.0 (app) · **Updated:** Jul 2026
**App users:** solo founders · vibe coders · indie hackers
**Output:** the one spoken CTA that ends a 30-sec promo (NOT tweet CTAs)
**Examples:** neutral — *InvoiceZen, a simple invoicing app for freelancers.*

---

## ⚠️ This folder was inverted

The original was built on organic-tweet data where **"no CTA wins."** That's wrong for a promo: a 30-second video is a pitch, so **every promo ends on exactly one CTA.** All files now reflect that.

## The One Rule

> Earn 22 seconds of attention, then spend the last 8 on **one** clear, low-friction step toward the outcome.

## Quick Reference
- **Always exactly one CTA** — never zero, never stacked.
- **Lowest-friction step** — "send your first invoice," not "choose a plan."
- **Names the outcome**, not the feature.
- **Intensity matches awareness** (`05`); **family matches offer type** (`06`).
- **Never fabricate urgency or scarcity.**
