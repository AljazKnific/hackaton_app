# WritingSystem-v2 — Index (App Script Craft)

The copywriting-craft layer: how each line of a promo script is actually written. Applied in Stage 3 after the angle, hook, structure, and tone are set.

**Version:** 2.0 (app) · **Updated:** Jul 2026
**App users:** solo founders · vibe coders · indie hackers
**Output:** polished spoken 30-sec promo copy (NOT tweets)
**Examples:** neutral — *InvoiceZen, a simple invoicing app for freelancers.*

## Quick Rules
- One idea per line, one point per promo.
- Clarity beats cleverness; plain words, 5th–8th-grade.
- Show, don't tell; recognition before the pitch.
- Honest only — no fabricated proof, fear, or scarcity.
- Written to be heard (delivery: `Voice-v2/07`).
