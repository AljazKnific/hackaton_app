# 01 — Script Craft Principles

The craft layer: how to write the actual *words* of a promo well, once you have the angle, hook, structure, and tone. This is the "make the copy good" pass.

**Role in the app:** applied while writing the Stage 3 script — turns a beat outline into sharp, spoken copy.

## The Core Principles

1. **One idea per line, one point per promo.** If a line carries two ideas, split it.
2. **Clarity beats cleverness.** 5th–8th-grade plainness. If the viewer has to re-parse a line, rewrite it.
3. **Show, don't tell.** Let the viewer reach the conclusion — "that makes sense" beats "here's what you should think."
4. **Make them feel seen before you pitch.** Recognition first, then the offer.
5. **Every line earns the next second.** No filler, no throat-clearing, no "so basically."
6. **Honest.** Real claims, real numbers, or none. Never manufacture proof or urgency.
7. **Written to be heard.** Short, spoken lines (see `Voice-v2/07` for delivery).

---

## The Quality Bar

| Level | Standard |
|---|---|
| Minimum | every line is clear and says one thing |
| Optimal | the viewer feels understood, then shifts how they see the problem |

If a script reads like an ad instead of a person talking, it fails — rewrite.
