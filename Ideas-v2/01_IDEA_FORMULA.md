# 01 — Idea Formula

How the app turns the marketing brain into **promo video angles** — the raw concepts that become hooks and scripts. This is the idea engine that sits between the brain and the script.

**Role in the app:** runs early in Stage 3. Produces a handful of distinct angles; the user (or the app) picks the strongest, which then becomes a hook (`ViralHooks-v2`) and a full script (`ICPOffer-v2/06_CONTENT_ANGLE_MAP`).

---

## The Core Formula

> **ICP Pain (or Desire) × Emotional Trigger × Reframe → Promo Angle**

1. **Pain / Desire** — pull one from the brain (`ICPOffer-v2/02`, `03`).
2. **Emotional trigger** — the feeling that makes it land (`02` in this folder).
3. **Reframe** — the twist that makes it feel fresh (`03` in this folder).
4. **→ Angle** — a one-line promo concept ready to be hooked and scripted.

---

## Where it sits in the pipeline

```
BRAIN (ICPOffer) → IDEAS (this folder): make 5 angles
     → pick the strongest
     → HOOK (ViralHooks): open the winning angle
     → SCRIPT (ICPOffer/06 beat map): build the 30-sec promo
     → VOICE (Voice-v2): set tone → ElevenLabs
```

---

## How many, and how varied

- Generate **~5 angles** per run so there's real choice.
- Vary them: different pains/desires, different triggers, different reframes — not five versions of one idea.
- Each angle must be **specific and honest** (real pain, real claim) — never generic filler.

---

## What a finished angle looks like

> **Angle:** "The invoice that pays itself" — freelancers stop chasing late payments because reminders send automatically.
> Pain: getting paid late · Trigger: relief · Reframe: Redefinition (the invoice does the work).

That one line is enough for the hook + script stages to build on.
