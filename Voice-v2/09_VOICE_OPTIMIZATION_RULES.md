# 09 — Voiceover Do / Don't

The spoken-delivery non-negotiables. Run every finished script through this before it goes to the voice model.

**Role in the app:** final voice QA gate before ElevenLabs.

---

## Always

| Rule | Why |
|---|---|
| Short lines, one breath each | the ear can't hold long sentences |
| Real, specific numbers | vague = forgettable; specific = believed |
| Speak to "you" | the promo is about the viewer |
| Numbers written as spoken words | the model reads them correctly |
| Contractions | sounds human, not stiff |
| One clear CTA at the end | a promo is a pitch |
| Confident, plain phrasing | trust comes from clarity |
| One tone mode throughout | consistent delivery |

---

## Never

| Rule | Why |
|---|---|
| Long, multi-clause sentences | unspeakable in one breath |
| Lowercase/punctuation "styling" for effect | invisible to the ear |
| Fabricated numbers, proof, or urgency | breaks trust; not allowed |
| Corporate jargon (synergy, leverage, paradigm) | dead on the ear |
| Stacked CTAs or no CTA | splits focus / wastes attention |
| Exclamation pile-ups | reads as desperate |
| Hedging the core claim | sounds unsure |
| Mixing tone modes mid-script | jarring delivery |

---

## The Voiceover Test

Read the script aloud. If you stumble, run out of breath, trip on a number, or it sounds like an ad instead of a person — rewrite before it ships to the voice model.
