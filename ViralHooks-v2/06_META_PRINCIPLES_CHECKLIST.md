# 06 — Meta-Principles + Hook Checklist

What makes all the hook patterns work — and the check every generated hook must pass before it ships into a script.

**Role in the app:** the quality gate between hook generation and the promo script. A hook that fails the checklist gets regenerated.

---

## The 6 Meta-Principles

1. **Specificity = credibility** — "11 days faster" beats "faster"; "3 hours a month" beats "saves time." Use the user's real numbers.
2. **Pattern interrupt** — the first spoken line breaks the expected opening.
3. **Self-recognition** — the viewer sees their exact situation; a hook for everyone lands for no one.
4. **Curiosity gap** — open a loop in the hook, close it later in the script — never in the first line.
5. **Stakes** — name what the problem is quietly costing them (time, money, credibility).
6. **Honest proof** — real metrics, real moments. Fabricated specificity is the one thing that breaks trust.

---

## Hook Checklist (before a hook enters a script)

- [ ] First spoken line violates the expected opening
- [ ] Contains at least one real, specific detail or number
- [ ] The viewer would think "that's me"
- [ ] Names a stake — the cost of not solving it
- [ ] Opens a loop the script resolves
- [ ] Can be said in one breath
- [ ] No ALL-CAPS, no read-only punctuation tricks
- [ ] Every number is real (from the user's input), not invented

**If it fails any of these — regenerate.**

---

## The Ultimate Truth

> A hook doesn't describe the product. It makes the viewer feel seen, then leaves them needing the rest.
> Every word earns the next second of attention.

---

## How this stacks with the rest of the system

Order of operations when generating a promo:
**pick the pain** (`ICPOffer-v2/02_PAIN_POINTS`) → **pick a hook pattern** (`02`–`05` in this folder) → **write the hook** (this checklist) → **build the full 30-sec script** (`ICPOffer-v2/06_CONTENT_ANGLE_MAP` beat map) → **voiceover pass** (spoken cadence). The pattern supplies the psychology; the beat map supplies the structure.
