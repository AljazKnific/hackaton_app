# ViralHooks-v2 — Index (App Hook Engine)

The hook library the app uses to write the **first 3 seconds of every promo** and shape the script around it. Feeds Stage 3 generation.

**Version:** 2.0 (app) · **Updated:** Jul 2026
**App users:** solo founders · vibe coders · indie hackers
**Output:** spoken promo-video hooks + scripts (NOT tweets)
**Examples:** one neutral product throughout — *InvoiceZen, a simple invoicing app for freelancers.*

## How the app uses this folder

1. Take the leading pain from `ICPOffer-v2/02_PAIN_POINTS`.
2. Pick a hook type here that fits the viewer's awareness stage.
3. Write the hook, run it through the `06` checklist.
4. Build the full 30-sec script on the `ICPOffer-v2/06_CONTENT_ANGLE_MAP` beat map.
5. Voiceover pass (`07` + spoken cadence).

---

## Two Non-Negotiables

- **Spoken, not written.** Hooks are heard — no ALL-CAPS, no read-only tricks; one breath per line.
- **Never fabricate.** Every number, story, and deadline must be the user's real data. Invented specificity is the one thing that breaks the whole system.
